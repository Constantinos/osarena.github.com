Κεντρικό Αποθετήριο Εφαρμογών του osarena.net

(Σελίς υπο κατασκευήν: version Beta )

To OSarena.net Apps αποτελεί το κεντρικό αποθετήριο λογισμικού απο το οποίο μπορείτε να κατεβάσετε ελεύθερα ανοιχτού κώδικα προγράμματα για τις Linux συσκευές σας. Για περισσότερα επικοινωνήστε μαζί μας στην ιστοσελίδα μας OSarena.net

FAQ ;; Τι εννοείτε ;

Το FAQ είναι η συντομογραφία του αγγλικού όρου Frequently Asked Questions. Ο σκοπός του FAQ είναι να συμπεριλάβει τα πιο συνήθη ερωτήματα που λαμβάνουμε και τις απαντήσεις τους.

Τι είναι το git και το zenity που ζητάτε να εγκαταστήσω ;

Το Git είναι ένα λογισμικό που αναλαμβάνει μας βοηθάει να διαχειριζόμαστε σε διαδικτυακό χώρο τα λογισμικά που ανεβάζουμε. Το Zenity το οποίο είναι συνήθως προεγκατεστημένο στο σύστημά σας αποτελεί ένα μινιμαλιστικό βοήθημα με το οποίο φτιάχνουμε γραφικά μηνύματα, παράθυρα και μενού επιλογών για τα προγράμματά μας.

Γιατί πρέπει να το εγκαταστήσω ;

Το πρόγραμμα Git πρέπει να το εγκαταστήσετε στο σύστημά σας για να μπορείτε να κατεβάζετε τις αποθήκες αλλά και να συγχρονίζετε στις τελευταίες εκδόσεις που είναι διαθέσιμες από τις αποθήκες μας. Το πρόγραμμα Zenity πρέπει να είναι εγκατεστημένο στο σύστημά σας για να μπορούν να εμφανίζονται τα μηνύματα, παράθυρα και τα μενού επιλογών των προγραμμάτων μας.

Πώς εγκαθιστώ το Git και το Zenity στο σύστημά μου ;

Ανοίξτε το Κέντρο Λογισμικού ή τον διαχειριστή πακέτων της διανομής σας, κάντε αναζήτηση τις λέξεις “Git” και “Zenity” και αν δεν είναι εγκατεστημένα, πατήστε εγκατάσταση.

Τις εφαρμογές ποιος τις έχει φτιάξει ; Είναι ασφαλείς ;

Οι περισσότερες εφαρμογές είτε φτιάχνονται από τους ίδιους τους υπεύθυνους του OSarena.net είτε προέρχονται από άλλους προγραμματιστές ελεύθερου λογισμικού. Οι εφαρμογές που βρίσκονται στο αποθετήριο μας ελέγχονται διεξοδικά, μεταφράζονται στα ελληνικά, δοκιμάζονται από τους υπεύθυνους του OSarena.net και αφού περάσουν κάποια αυστηρά κριτήρια ποιότητας, ανεβαίνουν στο αποθετήριο μας.

Πώς ενημερώνομαι αν έχετε βγάλει νέο λογισμικό ή έχετε αναβαθμίσει κάποιο από τα υπάρχοντα ;

Για να ενημερώνεστε για νέα λογισμικά ή αναβαθμίσεις των υπαρχόντων λογισμικών αρκεί να παρακολουθείτε τις εξελίξεις καθημερινά στην ιστοσελίδα μας

Πώς τις αναβαθμίζω η κατεβάζω τα νέα σας λογισμικά ;

Αφού ενημερώνεστε από την ιστοσελίδα μας, σε περίπτωση που υπάρξει κάποια ενημέρωση του αποθετηρίου ή μιας αποθήκης, εάν έχετε ήδη στο σύστημά σας την αποθήκη, το μόνο που χρειάζεται να κάνετε είναι να εκτελέσετε την εφαρμογή “update-repo” που θα βρείτε σε κάθε αποθήκη που έχετε κατεβάσει. Το “update-repo”, θα αναλάβει αυτόματα να κατεβάσει νέα λογισμικά ή και να αναβαθμίσει τα υπάρχοντα λογισμικά.

Έχω κατεβάσει μια από τις αποθήκες σας και θέλω να κατεβάσω και άλλη. Η υπάρχουσα αποθήκη θα διαγραφεί ;

Συμβουλές για το Αποθετήριο

Όταν εκτελέσετε τον εγκαταστάτη της αποθήκης που επιλέξατε, θα έχετε έναν φάκελο OSarenaRepo στον αρχικό σας κατάλογο
  • Εκεί μέσα θα έχετε τις αποθήκες που σας ενδιαφέρουν.
  • Να μην πειράζετε ή να διαγράφετε δεδομένα από μέσα αλλά ότι σας ενδιαφέρει να το πειράξετε/τρέξετε καλύτερα να το αντιγράφετε σε άλλο φάκελο της προτίμησης σας.
  • Ο φάκελος αυτός όταν θα εκτελείτε το update-repo θα διαγράφει ότι αλλαγές έχετε κάνει και θα εγκαθιστά τις τελευταίες εκδόσεις των περιεχομένων από τις αποθήκες.
  • Γενικά κρατήστε τον απείραχτο ώστε να είναι ο "εικονικός σύνδεσμος" σας με την αποθήκη του OSarena.
  • Όταν εμείς θα ανεβάζουμε ένα νέο λογισμικό το μονό που θα εκτελείτε είναι το πρόγραμμα update-repo και έτσι εκτός από αναβαθμίσεις, θα κατέβουν και νέα προγράμματα στους αντίστοιχους φακέλους.
