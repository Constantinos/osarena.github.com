Κεντρικό Αποθετήριο Εφαρμογών του OSArena.net

(Σελίς υπο κατασκευή: version Beta )

To OSArena.net Apps αποτελεί το κεντρικό αποθετήριο λογισμικού απο το οποίο μπορείτε να κατεβάσετε ελεύθερα ανοιχτού κώδικα προγράμματα για τις Linux συσκευές σας.

Aποθήκη Android Apps

Η αποθήκη αυτή περιλαμβάνει εφαρμογές, εργαλεία χρήσιμα για την καθημερινή χρήση της Android συσκευής σας. Ότι βρίσκεται σε αυτή την αποθήκη προορίζεται για εγκατάσταση στην Android συσκευή σας ή για τον υπολογιστή σας για να σας βοηθήσει με διάφορους τρόπους στη χρήση της. Σχετικά με το Andorid μπορείτε να μάθετε περισσότερα στην ιστοσελίδα μας OSarena.net

Οδηγίες χρήσης:

Πρίν κατεβάσετε τον εγκαταστάτη της αποθήκης Andrοid Apps εγκαταστήστε από το Κέντρο Λογισμικού της διανομής σας το πρόγραμμα Git και το Zenity.

Καντε κλίκ για να κατεβάσετε τον εγκαταστάτη της αποθήκης:

  • Αποσυμπιέζετε τo περιεχόμενο του συμπιεσμένου φακέλου στον αρχικό σας κατάλογο! (/home/user/).
  • Θα εμφανιστεί ένα πρόγραμμα με το όνομα της μορφής "download-*-Repo" και κάνετε διπλό κλικ για να ξεκινήσει η εγκατάσταση.
Θα δημιουργηθεί ο φάκελος OSarenaRepo μέσα στον οποίο θα υπάρχει ο φάκελος με την αποθήκη και όλα τα περιεχόμενα της διαδυκτιακής αποθήκης. Μέσα σε αυτόν τον φάκελο θα δείτε επίσης μια εφαρμογή "update-repo" την οποία μπορείτε να εκτελείτε όποτε θέλετε για να έχετε νέα λογισμικά (όταν θα είναι διαθέσιμα) και να αναβαθμίζεται τα υπάρχοντα λογισμικά. Παρακολουθείστε την ιστοσελίδα μας Osarena.net για να μαθαίνετε πότε και ποια νέα λογισμικά έχουν μπει στο αποθετήριο μας.

Συμβουλές για το Αποθετήριο

Όταν εκτελέσετε τον εγκαταστάτη της αποθήκης που επιλέξατε, θα έχετε έναν φάκελο OSarenaRepo στον αρχικό σας κατάλογο
  • Εκεί μέσα θα έχετε τις αποθήκες που σας ενδιαφέρουν.
  • Να μην πειράζετε ή να διαγράφετε δεδομένα από μέσα αλλά ότι σας ενδιαφέρει να το πειράξετε/τρέξετε καλύτερα να το αντιγράφετε σε άλλο φάκελο της προτίμησης σας.
  • Ο φάκελος αυτός όταν θα εκτελείτε το update-repo θα διαγράφει ότι αλλαγές έχετε κάνει και θα εγκαθιστά τις τελευταίες εκδόσεις των περιεχομένων από τις αποθήκες.
  • Γενικά κρατήστε τον απείραχτο ώστε να είναι ο "εικονικός σύνδεσμος" σας με την αποθήκη του OSarena.
  • Όταν εμείς θα ανεβάζουμε ένα νέο λογισμικό το μονό που θα εκτελείτε είναι το πρόγραμμα update-repo και έτσι εκτός από αναβαθμίσεις, θα κατέβουν και νέα προγράμματα στους αντίστοιχους φακέλους.

Μπορώ να βοηθήσω σε κάτι;

Υπάρχουν πολύ τρόποι να μας βοηθήστε αλλά οι σημαντικότεροι είναι:
  • Να μας ενημερώνετε αν βρείτε κάτι που δεν δουλεύει όπως θα έπρεπε.
  • Να μας ενημερώνετε αν κάποιο από τα προγράμματα μας δεν είναι πλήρως εξελληνισμένα
  • Να μας ενημερώνετε για μικρά εργαλεία, προγράμματάκια που θα θέλετε να δείτε στο αποθετήριο μας.
  • Αν έχετε να μας προτείνετε κάποια ιδέα που μας ξέφυγε επικοινωνήστε μαζί μας
  • Παρακολουθείστε την ιστοσελίδα μας OSarena.net για να μαθαίνετε πότε και ποια νέα λογισμικά έχουν μπει στο αποθετήριο μας
Για οποιαδήποτε απορία μπορείτε να μας αφήσετε μήνυμα στο τμήμα υποστήριξης του OSarena.net Repository Support
