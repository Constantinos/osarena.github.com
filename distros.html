Κεντρικό Αποθετήριο Εφαρμογών του osarena.net

(Σελίς υπο κατασκευήν: version Beta )

To OSarena.net Apps αποτελεί το κεντρικό αποθετήριο λογισμικού απο το οποίο μπορείτε να κατεβάσετε ελεύθερα ανοιχτού κώδικα προγράμματα για τις Linux συσκευές σας. Για περισσότερα επικοινωνήστε μαζί μας στην ιστοσελίδα μας OSarenae.net
Το Osarena.net εκτός από το ότι είναι το μεγαλύτερο ελληνικό site  που διαπραγματεύεται θέματα τεχνολογίας Linux και προσφέρει καθημερινά άρθρα/οδηγούς/ειδήσεις/ στους αναγνώστες του, σταδιακά άρχισε να επεκτείνει τις υπηρεσίες του παρέχοντας:
  • Βίντεο οδηγούς.
  • Λογαριασμούς αρθρογραφίας στις κοινότητες.
  • Συμμετοχή/δημοσιογραφική κάλυψη εκδηλώσεων των κοινοτήτων.
  • Android εφαρμογή OSarena.
  • Chrome εφαρμογή OSarena.
  • Δώρα και διαγωνισμούς για τους αναγνώστες μας
  • Κεντρικό Αποθετήριο Λογισμικών.
Ταυτόχρονα κάποιοι απο τους αρθρογράφους του, διαθέτουν και δικές τους διανομές. Έτσι το Osarena.net πέρα απο τις διάφορες διανομές Linux για τις οποίες γράφει καθημερινά διαθέτει και επίσημες Διανομές/Projects φτιαγμένα από τους αρθρογράφους του.

Ρίξτε μια ματιά και επιλέξτε ανάλογα με τις ανάγκες σας για να τα εγκαταστήσετε σον υπολογιστή σας ως το καθημερινό λειτουργικό σας σύστημα:

Τι είναι Αποθετήριο;

Φανταστείτε το αποθετήριο ως μια διαδικτυακή αποθήκη λογισμικού, εργαλείων, κειμένων και άλλων τύπων δεδομένων από την οποία κατεβάζετε όλα τα δεδομένα που σας ενδιαφέρουν . Αυτή τη στιγμή είναι διαθέσιμες 4 αποθήκες λογισμικού ανάλογα με τις προτιμήσεις σας. Μπορείτε να χρησιμοποιείσετε όσες και όποιες θέλετε. Διαβάστε περισσότερα >>

Μπορώ να βοηθήσω σε κάτι;

Υπάρχουν πολύ τρόποι να μας βοηθήστε αλλά οι σημαντικότεροι είναι:
  • Να μας ενημερώνετε αν βρείτε κάτι που δεν δουλεύει όπως θα έπρεπε.
  • Να μας ενημερώνετε αν κάποιο από τα προγράμματα μας δεν είναι πλήρως εξελληνισμένα
  • Να μας ενημερώνετε για μικρά εργαλεία, προγράμματάκια που θα θέλετε να δείτε στο αποθετήριο μας.
  • Αν έχετε να μας προτείνετε κάποια ιδέα που μας ξέφυγε επικοινωνήστε μαζί μας
  • Παρακολουθείστε την ιστοσελίδα μας Osarenae.net για να μαθαίνετε πότε και ποια νέα λογισμικά έχουν μπει στο αποθετήριο μας
Για οποιαδήποτε απορία μπορείτε να μας αφήσετε μήνυμα στο τμήμα υποστήριξης του Osarena.net Repository Support
